setInputs(inputs, isInputLayer)
Sets the inputs for all the Neurons in this Layer.
Parameters:
{Array} inputs
An array of inputs, which could be numbers, functions returning a number or Neurons.
{boolean} isInputLayer
Indicates whether this is intended to be the input layer in a Network. If set to true, rather than each of the supplied inputs being wired into each of the Neurons in this Layer, they will be mapped one-to-one, with no bias (i.e. passed straight through to the next layer).
